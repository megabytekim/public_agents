# 기업분석 명령어

**목적**: Market Intelligence(MI)와 Portfolio Intelligence(PI) 에이전트를 협업시켜 종합적인 기업 분석을 수행합니다.

## 실행 워크플로우

### Phase 1: 데이터 수집 (MI 담당)

Market Intelligence 에이전트를 호출하여 다음을 수집합니다:

```
MI 태스크:
1. 오늘 날짜 확인 (필수)
2. 실시간 주가 확인 (yfinance MCP 또는 WebFetch)
3. 최신 뉴스 수집 (WebSearch)
4. 재무 데이터 수집 (Playwright - FnGuide/Yahoo Finance)
5. 애널리스트 의견 수집
```

**MI 호출 프롬프트**:
```
[ticker] 종목에 대해 다음 정보를 수집해주세요:

1. 현재 주가 및 변동률 (반드시 출처와 시간 명시)
2. 최근 1주일 주요 뉴스 3-5개
3. 주요 재무지표 (PER, PBR, ROE, 매출/영업이익 추이)
4. 52주 최고/최저, 거래량 분석
5. 최근 애널리스트 목표가 및 투자의견

모든 데이터는 반드시 출처와 확인 날짜를 명시해주세요.
```

### Phase 2: 전략적 분석 (PI 담당)

Portfolio Intelligence 에이전트를 호출하여 다음을 분석합니다:

```
PI 태스크:
1. 섹터 포지셔닝 분석
2. 경쟁사 대비 강점/약점
3. 투자 테제 수립
4. 리스크 요인 식별
5. 진입/청산 전략 수립
```

**PI 호출 프롬프트**:
```
MI가 수집한 [ticker] 데이터를 바탕으로 전략적 분석을 수행해주세요:

1. 해당 기업의 섹터 내 포지셔닝
2. 경쟁 우위 (Moat) 분석
3. 단기/중기/장기 투자 테제
4. 주요 리스크 요인 3가지
5. 추천 진입가/목표가/손절가

Obsidian에 저장할 경우 경로도 제안해주세요.
```

## 분석 깊이별 차이

### quick (빠른 분석)
- MI: 가격 + 뉴스 2개
- PI: 간단한 의견
- 소요시간: ~2분

### standard (표준 분석)
- MI: 가격 + 뉴스 5개 + 재무지표
- PI: 섹터 분석 + 투자 전략
- 소요시간: ~5분

### deep (심층 분석)
- MI: 전체 데이터 + 경쟁사 비교 + 과거 이벤트
- PI: 상세 밸류에이션 + 시나리오 분석 + Obsidian 저장
- 소요시간: ~10분

## 사용 예시

```bash
# 기본 사용
/stock-analyzer-advanced:기업분석 NVDA

# 깊이 지정
/stock-analyzer-advanced:기업분석 NVDA deep

# 한국 주식
/stock-analyzer-advanced:기업분석 삼성전자
/stock-analyzer-advanced:기업분석 005930
```
