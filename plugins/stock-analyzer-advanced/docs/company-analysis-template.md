## PI 에이전트 사용 플로우 (업데이트)

```
1. 사용자가 경로 제공
   ↓
2. PI가 obsidian_get_file_contents() 호출
   ↓
3. 기존 내용 파악
   ↓
4. ⚠️ STEP 0: 오늘 날짜 확인 (필수)
   WebFetch("https://www.google.com", "오늘 날짜")
   ↓
5. MI에게 최신 데이터 요청
   ↓
6. STEP 1: yfinance MCP로 가격 검증 (최우선)
   또는 STEP 2: WebFetch Yahoo Finance
   ↓
7. STEP 3: WebSearch로 뉴스 수집 (날짜 포함)
   ↓
8. STEP 4: Playwright로 차트 확인 (선택)
   ↓
9. PI가 섹터 지식 결합
   ↓
10. obsidian_append_content()로 업데이트 추가
    (날짜 + 출처 명시)
   ↓
11. 사용자에게 완료 알림
```

## 데이터 수집 프로토콜

### 미국 주식 분석 시
```bash
# STEP 0: 날짜 확인
WebFetch("https://www.google.com", "오늘 날짜")

# STEP 1: yfinance MCP (최우선)
mcp__yfinance__get_stock_price(ticker="NVDA")

# STEP 2: WebFetch (MCP 없을 시)
WebFetch("https://finance.yahoo.com/quote/NVDA",
         "현재 주가, 날짜 추출")

# STEP 3: WebSearch (뉴스)
WebSearch("NVDA stock news December 30 2025")

# STEP 4: Playwright (차트)
browser_navigate("https://finance.yahoo.com/quote/NVDA")
```

### 한국 주식 분석 시
```bash
# STEP 0: 날짜 확인
WebSearch("오늘 날짜")

# STEP 1: FnGuide (필수)
browser_navigate("https://comp.fnguide.com/...")

# STEP 2: WebSearch (뉴스)
WebSearch("케이옥션 뉴스 2025년 12월 30일")

# STEP 3: Naver Finance
WebSearch("케이옥션 실시간 시세")
```
