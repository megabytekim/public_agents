# Tier 2 Analyzer Plugin

> stock-analyzer-advanced의 Tier 2 심층 분석 확장

## 개요

Tier 1 (`stock-analyzer-advanced`)의 1차 스크리닝 후, **심층 펀더멘털 분석**을 수행하는 에이전트 시스템.

## 아키텍처

```
┌─────────────────────────────────────────────────────────────────┐
│                         TIER 1 (스크리닝)                        │
│                    plugins/stock-analyzer-advanced               │
├─────────────────────────────────────────────────────────────────┤
│    ┌────────┐    ┌────────┐    ┌────────┐    ┌────────┐        │
│    │   MI   │    │   SI   │    │   TI   │    │   FI   │        │
│    │ (뉴스) │    │(종토방)│    │(기술적)│    │(연간)  │        │
│    └────────┘    └────────┘    └────────┘    └────────┘        │
└─────────────────────────────────────────────────────────────────┘
                              │
                              │ Tier 1 분석 완료 후
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│                         TIER 2 (심층분석)                        │
│                    plugins/tier2-analyzer                        │
├─────────────────────────────────────────────────────────────────┤
│    ┌──────────────┐   ┌──────────────┐   ┌──────────────┐      │
│    │     FI+      │   │     SI+      │   │     MI+      │      │
│    ├──────────────┤   ├──────────────┤   ├──────────────┤      │
│    │ 분기 재무    │   │ 텔레그램     │   │ 경영진 평가  │      │
│    │ + 동종비교   │   │ 센티먼트     │   │ + DART 공시  │      │
│    └──────────────┘   └──────────────┘   └──────────────┘      │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
                    tier2/{종목명}_{코드}/
                        └── summary.md
```

## 에이전트

| 에이전트 | 역할 | 소스 |
|----------|------|------|
| **FI+** | 분기 재무 + 피어 비교 | FnGuide, DART |
| **SI+** | 텔레그램 센티먼트 | Telethon |
| **MI+** | 경영진 평가 + DART 공시 | OpenDART API |

## 커맨드

| 커맨드 | 설명 |
|--------|------|
| `/deep-analyze` | Tier 2 심층 분석 실행 |

## 출력 경로

```
tier2/
├── 서진시스템_178320/
│   └── summary.md
├── 카카오페이_377300/
│   └── summary.md
└── ...
```

## 의존성

- Tier 1: `stock-analyzer-advanced` 플러그인
- Python: pykrx, requests, beautifulsoup4, telethon
- API: OpenDART API, Telegram API
